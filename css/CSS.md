# CSS

## 1. 선택자와 선언

- Selector: 선택자
- Declaration: 선언
  > - property(속성), value(속성 값), separator(구분자) 존재

### 1.1 선택자의 종류

- 태그 선택자: 태그를 선택하는 선택자
- 아이디 선택자: 해당 아이디를 선택
  > ex) #select{} :아이디값이 select에 한해 해당태그에 대해서 속성값을 가짐
- 클래스 선택자: 모든 선택자 중에서 클래스값이 해당되는 태그에 한해 속성이 적용
  > ex) .클래스의 태그
  >
  > - id는 단 한번만 등장(같은 태그만)/class는 해당하는 것을 그룹핑하는 느낌(다른 태그)
