# HTML

## 1. 기본문법- 태그

### 1.1 태그

```
- '<h1> ...</h1>' ~ '<h6>...</h6>': 제목으로 처리
- '<strong>...</strong>': 진하게 표시
- '<a href="url"> ... </a>': 링크를 연결
- '<li>...<li>': 각각의 항목 구분
- '<ul>...</ul>': 성격이 같은 항목들을 묶어 리스트를 감싸는 태그(그룹핑)
- '<ol>...</ol>': 순서가 있는 리스트를 감싸는 태그(그룹핑)
- '<title>...</title>': 웹사이트 제목,  `<head>` 내에 쓰는 것을 권장
- '<meta charset="utf-8">': 한글깨짐현상 없앰, 한글 디코딩, `<head>` 내에 쓰는 것을 권장
- '<head>...</head>':본문이 아닌 태그, 문서를 설명하는 태그
- '<body>...</body>':본문인 태그
- '<html>...</html>': 가장 바깥쪽에 위치하는 태그
- '<!DOCTYPE html>':어떠한 표준을 따르는 태그인지 브라우저에게 알려줌
- '<p>...</p>': 단락을 나타내는 태그, 줄바꿈간격이 고정
- '<br>': 줄바꿈, 열린태그만 존재
- '<img src="사진파일">': 이미지 삽입 태그
- '<td>...</td>': 테이블의 각각의 항목
- '<tr>...</tr>': td를 같은 행끼리 그룹
- '<table>...</table>': 표 만들기


```

### 1.2 form 태그

```
- <input type="..", name="입력값의 변수명">: 사용자가 정보입력할 수 있는 태그
- <form action="url(서버 주소)">...</form>: 사용자가 입력한 정보를 제출(submit)을 통해 어느 서버로 보낼 건지
- <textarea></textarea>:여러줄을 입력할 수 있는 태그
- <option>...</option>:옵션 태그
- <select name="">...</select>: 옵션을 선택할 수 있게 dropdowmlist를 만드는 태그
- <input type="radio",name="x">: inpout값이 동일한 name값끼리 그룹핑
- <input type="checkbox",name="x">: inpout값이 동일한 name값끼리 그룹핑, 다중 선택 가능
- <label for = "누구의 이름표인지">...</label>: 이름표 태그
```

### 1.3 의미론적인 태그

```
- '<footer>...</footer>': 아래에 위치시키는 태그
- '<nav>...</nav>': 컨텐츠를 탐색할 때 위치를 알려주는 태그
- '<article>...</article>': 본문을 나타내는 태그
- '<section>...</section>': 역할이 분명하지 않은 것을 묶어주는 의미론적 태그
```
